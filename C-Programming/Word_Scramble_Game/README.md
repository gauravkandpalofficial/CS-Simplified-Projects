# 🎮 Word Scramble Game

> A beginner-friendly console-based Word Scramble Game built using **C Programming**.

This project is part of my **CS-Simplified Projects** collection, where I build practical projects while learning and applying Computer Science concepts from scratch.

---

# 📌 Project Overview

The **Word Scramble Game** is a console-based application that challenges players to identify the correct word from a scrambled version.

Players can choose different difficulty levels, use hints whenever required, and earn points based on their performance.

The primary objective of this project is to strengthen programming fundamentals while making learning enjoyable and interactive.

---

# ✨ Features

* 🎯 Three Difficulty Levels

  * Easy
  * Medium
  * Hard

* 💡 Hint System

* 🏆 Score Tracking

* 🔤 Random Scrambled Words

* ✅ Case-Insensitive Answer Validation

* 📚 Beginner-Friendly Code

* 💻 Console-Based Interface

# 🚀 How to Run

### Clone the repository

```bash
git clone <repository-link>
```

### Open the project folder

```bash
cd Word_Scramble_Game
```

### Compile

```bash
gcc word_scramble_game.c -o game
```

### Run

```bash
./game
```
